# kun-nuxt-template

A mini starter for Nuxt. Just for concise and moe!

![](docs/banner.png)


## How to use

Click `Use this template` to create your own repository, clone your repository, open it and input

``` bash
pnpm i && pnpm dev
```

to your terminal.
